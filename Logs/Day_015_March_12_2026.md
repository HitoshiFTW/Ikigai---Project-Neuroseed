## 2. Part 1 — LC-NE Threat Coupling

### 2.1 Biological Basis

The locus coeruleus norepinephrine (LC-NE) system is the brain's primary arousal and gain-control mechanism (Aston-Jones & Cohen 2005, Adaptive Gain Theory). Crucially, LC neurons receive projections from multiple threat-relevant sources: the amygdala (BLA→LC), the HPA axis (corticotropin-releasing hormone→LC), nociceptive afferents, and ascending cortical prediction error signals (Sara 2009, Arnsten 2012). The prior implementation responded only to cortical prediction error.

### 2.2 Implementation

`NorepinephrineSystem.update()` now accepts four inputs:

| Input | Biological Source | Weight |
|---|---|---|
| `signal` | Cortical prediction error / novelty | 0.50 |
| `cortisol_level` | HPA axis stress signal to LC | 0.25 |
| `pain_signal` | Nociceptive LC afference | 0.15 |
| `amygdala_threat` | BLA→LC projection (`amyg.bla_valence`) | 0.10 |

```python
ne_drive = signal*0.50 + cortisol_level*0.25 + pain_signal*0.15 + amygdala_threat*0.10
delta = abs(ne_drive - self.last_sig)
if delta > 0.25:           # spike threshold (tightened from 0.30)
    self.level += 0.15     # spike amplitude (reduced from 0.20)
```

The main loop passes `cort.level`, `1.0 if env.pain_sudden else 0.0`, and `max(0.0, amyg.bla_valence)`.

## 3. Part 2 — Predictive Sleep System

### 3.1 Biological Basis

Borbély (1982) two-process model: sleep pressure (Process S) accumulates as a function of prior wakefulness. Adenosine is the molecular substrate of Process S (Porkka-Heiskanen et al. 1997). Friston (2010) active inference reframes this: the organism predicts future fatigue and initiates sleep anticipatorily to minimize the surprise of collapse-driven sleep onset. Tononi & Cirelli (2014) synaptic homeostasis provides the energy depletion amplification term.

### 3.2 Implementation — `PredictiveSleepSystem`

EMA tracking of both adenosine and energy (α = 0.02, slow window τ ≈ 50 ticks):

```
adenosine_trend = α × adenosine + (1−α) × adenosine_trend
energy_trend    = α × avg_energy + (1−α) × energy_trend

PSP = adenosine_trend + 0.60 × (1 − energy_trend)
```

- **Sleep onset**: PSP > 0.65 (anticipatory — lower than homeostatic 0.70)
- **Sleep end**: adenosine < 0.20 (Borbély clearance)
- **Merge**: `sleep_drive = max(homeostasis.sleep_drive, PSP)` — whichever is stronger triggers sleep

```
PRED_SLEEP: pressure=0.31  ado_trend=0.22  e_trend=0.71
```

## 4. Part 3 — Spatial Cognition System

### 4.1 Biological Basis

O'Keefe & Nadel (1978): hippocampal place cells fire at specific spatial locations. Hafting et al. (2005): entorhinal grid cells fire in hexagonal lattices across the environment. Moser et al. (2008): together these form the brain's GPS system.

### 4.2 Implementation — `SpatialNavigationSystem`

**Grid cells** (8 cells): Hexagonal activation via three equally-spaced cosine projections:
```
G_k(x,y) = ½ [cos(f·proj₀) + cos(f·proj₆₀) + cos(f·proj₁₂₀)]
```

**Place cells** (12 cells): Gaussian firing fields:
```
PC_i(x,y) = exp(−((x−xᵢ)² + (y−yᵢ)²) / 2σ²),  σ = 2.5 units
```

**Position dynamics**: ±0.35 units/tick from action selection (approach/withdraw/explore), bounded ±100.

**Spatial novelty**: `1 − familiarity`, where familiarity is a weighted combination of place cell activation and visit count. When `novelty > 0.4`:
```
curiosity_drive += min(0.20, (novelty − 0.40) × 0.40)
```

```
SPATIAL: pos=(2.4,−1.1)  place=0.48  grid=0.31  novelty=0.47
```

## 5. Part 4 — Episodic Replay System

### 5.1 Biological Basis

During SWS sleep, CA3 place cell sequences active during waking are reactivated in temporally compressed sequences (replay speed ~10× waking, Wilson & McNaughton 1994). Sharp-wave ripples (SPW-Rs) coordinate this hippocampal replay (Buzsáki 2015, Girardeau et al. 2009). Replay preferentially activates emotionally salient memories (Stickgold 2005). Sleep-dependent emotional regulation during replay attenuates the affective response to fearful memories (Walker & van der Helm 2009).

### 5.2 Implementation — `EpisodicReplaySystem`

**Trajectory buffer**: FIFO deque of `(x, y, valence, cortisol, curiosity)` tuples during wake, max 200 entries.

**Replay trigger** (SWS-gated):
```
P_replay = α × sleep_depth   (α = 0.05)
```

**Weighted sequence selection** — emotionally salient memories replay more often:
```
P(seq) ∝ |valence| + cortisol   per sequence window
```

**Emotional attenuation**:
```python
if fear_trace > 0.6:
    fear_trace *= 0.95   # Walker & van der Helm 2009
```

**Dream token generation**: Spatial-emotional summaries of replay sequences injected into `dream_sys.dream_log`:
- High cortisol: `"i remember being far. fear was smaller now."`
- Positive valence: `"i was close and it felt good."`
- Negative valence: `"i drifted far. something was wrong."`
- Neutral: `"i was close. calm."`

Max 5 replays per sleep cycle. Sleep counter resets on sleep onset.

```
REPLAY: seqs=3  avg_len=9.2  emotional=2  buf=147
  dream: "i remember being close. fear was smaller now."
```

## 6. Part 5 — Forward Planning System

### 6.1 Biological Basis

During wakeful deliberation pauses (behavioral pauses, theta sequences), the hippocampus performs forward sweeps simulating possible future trajectories (Pfeiffer & Foster 2013). The prefrontal cortex evaluates these paths using learned value estimates (Johnson & Redish 2007). Tolman (1948) demonstrated that rats use cognitive maps for goal-directed navigation without directly experiencing all paths.

### 6.2 Implementation — `PlanningSystem`

**Trigger conditions** (refractory: 1 event per 5 ticks):
- `selected_action == 'SUPPRESSED'` (motor idle), OR
- `prediction_error > 0.3` (high uncertainty)

**Forward simulation**: 4 candidate paths, each 7 steps of 0.20 spatial units in random directions.

**Value function**:
```
V = 0.40 × C + 0.40 × R − 0.20 × S

C = mean spatial novelty along path  (curiosity)
R = mean cognitive map reward estimate
S = current cortisol level           (stress penalty)
V ∈ [−1, 1]
```

**Motor bias**: `explore_drive += best_path_value × 0.20` (bounded +0.30), applied to homeostasis curiosity drive. Decays 0.02/tick during non-planning ticks.

```
PLANNING: paths=4  best_V=0.42  events=87  bias=0.08
```

## 7. Part 6 — Prefrontal Cognitive Map System

### 7.1 Biological Basis

Tolman (1948) demonstrated that rats acquire internal cognitive maps allowing flexible navigation. Behrens et al. (2018) showed that prefrontal cortex encodes abstract relational maps generalising beyond physical space. Each visited location accrues a reward estimate updated by experience (temporal difference learning, O'Doherty et al. 2003).

### 7.2 Implementation — `CognitiveMapSystem`

**Node indexing**: 2D spatial grid at resolution 2.0 units. Arena ±100 → 10,000 possible cells.

**Node structure**: `{visits: int, reward: float}` per cell.

**Reward update** (EMA, α = 0.15):
```
reward = 0.15 × valence + 0.85 × reward,   reward ∈ [−1, 1]
```

**Explored ratio**: `len(nodes) / 10,000`

**Planning integration**: `get_reward(x, y)` provides the R term in PlanningSystem's value function. Unvisited cells return 0 (novelty-seeking is handled via the C term).

```
MAP: nodes=87  explored=0.0087  visits=1000
```
