## 3. Architecture Introduced

A new `CorticalColumn` system was implemented. The cortex is now specifically composed of layered neuron populations functionally mirroring biological columns.

### 3.1 Layer 4 (L4)
- **Role:** Sensory Input Layer
- **Scale:** ~30 neurons
- **Function:** Receives raw environment vectors. It features minimal recurrence and acts as the entry point for granular sensory representations.

### 3.2 Layer 2/3 (L23)
- **Role:** Associative Cortex
- **Scale:** ~60 neurons
- **Function:** Features a dense recurrent attractor network (~15% local connectivity). This layer generates internal representations, maintains traces, drives the prediction system, and interfaces deeply with the hippocampus.

### 3.3 Layer 5 (L5)
- **Role:** Motor/Output Cortex
- **Scale:** ~30 neurons
- **Function:** Converts heavily processed and filtered activity from L2/3 into concrete behavioral actions.

### 3.4 Local Inhibitory Pool
- **Scale:** ~15 interneurons
- **Function:** Operates as a shared inhibitory microcircuit. It continuously tracks excitation and enforces a biologically realistic winner-take-all (WTA) competition to selectively sharpen network activity.

## 4. Predictive Processing System

A rigorous predictive coding mechanism was explicitly integrated based on Karl Friston's predictive models @[Digital Organism Neuroscience Research Synthesis.txt]. 

L2/3 acts as a predictive engine continuously attempting to predict the incoming sensory state of L4. Prediction is generated mathematically through a structured prediction matrix (60 × 30).

Prediction error is mathematically computed as:
`error = actual_L4 - predicted_L4`

Crucially, **L4 activity is never modified by prediction.** The generative prediction error strictly serves to update the prediction weights in the matrix. 

Error-driven learning is continuously gated and modulated by the biological state:
- **Dopamine:** Scales the learning rate, representing reward prediction error @[Digital Organism Neuroscience Research Synthesis.txt].
- **Norepinephrine:** Scales prediction error precision corresponding to exploration-exploitation arousal @[Digital Organism Neuroscience Research Synthesis.txt].

## 5. Cortical Processing Order

A strict biological processing order was implemented inside the synchronous simulation tick loop to ensure inhibition acts appropriately on stabilized representations.

1. **Step 1:** Environment generates sensory vector.
2. **Step 2:** L4 neurons receive sensory input and spike.
3. **Step 3:** L4 spikes propagate to L2/3.
4. **Step 4:** L2/3 performs recurrent attractor settling for 3 cycles.
5. **Step 5:** Local inhibitory microcircuit enforces winner-take-all competition.
6. **Step 6:** Prediction system computes prediction error between L2/3 and L4.
7. **Step 7:** Hippocampal interaction occurs (CA3 encoding or CA1 retrieval).
8. **Step 8:** L2/3 propagates signals to L5 motor neurons.
9. **Step 9:** Motor output triggers behavior if the threshold is reached.

## 6. Oscillatory Modulation

We successfully integrated global brain rhythms across the entire cellular population to coordinate spike timing organically without forcing mathematical synchronization.

- **Theta Rhythm:** ~4-8 Hz analogue equivalent, ±0.08 amplitude.
- **Gamma Rhythm:** ~40 Hz analogue equivalent, ±0.04 amplitude.

These oscillations globally modulate neuronal firing thresholds, guiding neurons into unified firing sequences critical for tasks like memory replay.
