# Ikigai, Project NeuroSeed

A biologically grounded digital organism exploring how cognition, language, and autobiographical continuity can emerge from interacting neural systems rather than explicit symbolic programming.

---

## Overview

NeuroSeed is an experimental artificial life and computational neuroscience research project focused on **emergent cognition from biologically motivated mechanisms**.

Rather than training on datasets or hardcoding behaviors, Ikigai develops internal structure through:

- excitation–inhibition balance
- neuromodulator dynamics
- hormonal stress regulation
- homeostatic drives
- latent state compression
- replay and sleep consolidation
- concept abstraction
- grounded language readout
- autobiographical narrative memory

The core scientific question is:

> Can persistent cognition, self-generated language, and reflective continuity emerge from a compact biologically inspired organism running on minimal compute?

This project is the experimental substrate for that question.

## Core Biological Systems

### Neural Dynamics

Neurons use a simplified biologically inspired leaky-integrator framework with:

- threshold firing
- refractory periods
- calcium fatigue
- intrinsic threshold adaptation
- homeostatic plasticity

---

### Excitation–Inhibition Balance

Network stability is maintained through:

- inhibitory plasticity
- adaptive thresholds
- population-scaled inhibitory influence
- oscillatory balance pressure

Inspired by:

- Vogels et al. (2011)
- Turrigiano (1998)

---

### Neuromodulators

The organism models interacting simplified neuromodulatory systems:

| System | Role |
|---|---|
| Dopamine | reward, motivation, action reinforcement |
| Norepinephrine | arousal, gain control, uncertainty response |
| Acetylcholine | cortical coordination, task focus |
| Oxytocin | social buffering, trust, stress reduction |
| Cortisol | stress dynamics and HPA-axis pressure |

These continuously influence firing dynamics, confidence, replay salience, and behavioral stability.

The organism includes survival-style internal regulation:

- energy balance
- hunger drive

- curiosity drive
- uncertainty pressure
- metabolic restoration

This creates biologically grounded motivation without symbolic goal scripting.

## Emergent Cognition Stack

The current cognition substrate is layered as a causal hierarchy.

### Foundation Cognition

- **LatentStateVector** → 64D compressed lived present
- **ActionReasoningLog** → per-tick rationale continuity
- **ReplayBuffer** → salience-biased transition memory
- **EventCompressor** → multi-tick episode abstraction
- **ConceptGraph** → stable reusable concept motifs

### Workspace and Language

- **ReportBus** → unified semantic workspace
- **LanguageReadout** → meaning-space propositions
- **SentenceGenerator** → grounded self-generated English sentences

### Offline Continuity

- **SleepConsolidator** → sleep-based semantic stabilization
- **NarrativeMemory** → autobiographical memory arcs

This stack allows the organism to move from:

> lived state → concepts → language → sleep replay → self-history
